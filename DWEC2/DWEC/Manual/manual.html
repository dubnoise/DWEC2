MANUAL DE ETIQUETAS HTML

Etiqueta 'h1', 'h2', 'h3', 'h4', 'h5' y 'h6' - Se utiliza para títulos o subtítulos.

EJEMPLO:

<h2>Esto es un título.</h2>

RESULTADO:

Esto es un título.

Etiqueta 'p' - Se utiliza para escribir parágrafos.

EJEMPLO:

<p>Esto es un parágrafo.</p>

RESULTADO:

Esto es un parágrafo.

Etiqueta 'a' - Establece un vínculo.

EJEMPLO:

<a>Esto es un vínculo.</a>

En esta etiqueta se suele utilizar el atributo 'href' para la dirección del vínculo.

<a href="Aquí está la dirección" Wikipedia</a>

RESULTADO:

Wikipedia

Etiqueta 'img' - Se utiliza para insertar imágenes.

Se suele utilizar los atributos 'src' para establecer qué images se va a insertar; y 'alt' para indicar el texto alternativo de la imagen.

EJEMPLO:

<img> src="Aquí está la url de la imagen" alt="Aquí está el texto alternativo">

RESULTADO:

Etiqueta 'br' - Se utiliza para un salto de línea.

EJEMPLO:

<p>Esto es un salto <br> de línea.</p>

RESULTADO:

Esto es un salto
de línea.

Etiqueta 'form' - Se utiliza para formularios.

Dentro de 'form' podemos poner las siguientes etiquetas:

EJEMPLO:

<form>

    <label>Nombre:</label>

    <input type="text">

    <label>Apellido:</label>

    <input type="text">

    <input type="submit" value="Enviar">

</form>

RESULTADO:

Etiqueta 'b' - Se utiliza para poner el texto en negrita.

EJEMPLO:

<p>Este texto NO está en negrita <b>Este texto SI está en negrita</b></p>

RESULTADO:

Este texto NO está en negrita Este texto SI está en negrita

Etiqueta 'i' - Se utiliza para poner el texto en cursiva.

EJEMPLO:

<p>Este texto NO está en cursiva <i>Este texto SI está en cursiva</i></p>

RESULTADO:

Este texto NO está en cursiva Este texto SI está en cursiva

Etiqueta 'ul' - Se utiliza para definir una lista desordenada.

Para definir un elemento de la lista se utiliza 'li'.

EJEMPLO:

<ul>

    <li>Elemento 1</li>

    <li>Elemento 2</li>

    <li>Elemento 3</li>

</ul>

RESULTADO:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Etiqueta 'table' - Se utiliza para crear tablas.

La etiqueta 'tr' define una fila en la tabla

La etiqueta 'td' define una celda de datos en la tabla

EJEMPLO:

<table>

    <tr>

        <td>Celda A</td>

        <td>Celda B</td>

    </tr>

    <tr>

        <td>Celda C</td>

        <td>Celda E</td>

    </tr>

</table>

RESULTADO:

Celda A Celda B
Celda C Celda E

Etiqueta 'u' - Se utiliza para subrayar texto.

EJEMPLO:

<p>Este texto NO está subrayado <u>Este texto SI está subrayado</u></p>

RESULTADO:

Este texto NO está subrayado Este texto SI está subrayado

Etiqueta 'html' - Representa la raíz de un documento HTML

Esta etiqueta es el contenedor de todos los demás elementos HTML

Etiqueta 'head' - Es un contenedor de metadatos.

Se coloca entre la etiqueta 'html' y la etiqueta 'body'

Etiqueta 'body' - Parte visible en pantalla del documento.

Se coloca después del 'head'
